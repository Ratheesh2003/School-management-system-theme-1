About This Project

Student Result Management System Should Be Abel to Log All Students Data, There Marks, There Year Of Studies, There Attendance..(You Can Create Your Own Dataset Having Columns Student ID, Student Name,Student Course, Student Marks, Student Attendance) It Should Also Have A Score Board Where The Students Scoring Maximum Marks Will Be Highlited And Maximum Attendance Percentage It Should Have Login System Using Javascript, Once Logged In They Should Be Able To Add A New Record To The Database
